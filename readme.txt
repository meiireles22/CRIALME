-Descrição do projeto

#Este projeto consiste numa loja online desenvolvida para uma empresa especializada na venda de fatos e fatos por medida. 

#O sistema permite aos clientes navegar pelos produtos, personalizar fatos e realizar encomendas online. 

-Tecnologias Utilizadas

#Frontend: HTML, CSS, JavaScript

#Backend: PHP

#Base de Dados: MySQL (MySQL Workbench)

-Funcionalidades:

#Catálogo de produtos

#Personalização de fatos por medida;

#Integração com base de dados;

#sistema de loggin;


-Instalação e Configuração

#Clone este repositório:

#git clone <URL_DO_REPOSITORIO>

#Configure a base de dados utilizando o ficheiro SQL incluído  no MySQL Workbench

#Edite o ficheiro de conexão com a base de dados para inserir as credenciais corretas.

#Execute o servidor local (XAMPP).

#Acesse a aplicação via navegador.
